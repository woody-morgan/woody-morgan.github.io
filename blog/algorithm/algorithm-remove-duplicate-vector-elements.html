[Algorithm] Remove Duplicate Vector Elements

Authors
  • Name
    Kim, Dong-Wook
    Twitter
Table of Contents

Intro

While solving BOJ 18870 Problem, I found a very easy way to remove duplicated c++ vector elements.

#include <algorithm>
#include <iostream>
#include <vector>

using namespace std;

void print(vector<int>& v) {
  for (const auto& val : v) {
    cout << val << " ";
  }
  cout << "\n";
}

int main(void) {
  vector<int> v = {1, 1, 1, 5, 4, 2, 3, 1, 1};

  unique(v.begin(), v.end());
  print(v);
  // result : 1, 5, 4, 2, 3, 1, 3, 1, 1

  return 0;
}
  1. We can remove duplicated elements by using unique function.
  2. But, Before use unique function, you should sort the vector first.
  3. Because unique function remove duplicated elements by comparing the current element and the next element.
  4. If the next element is same with the current element, unique function throws the next element to the end of the vector.
  5. And unique function returns the iterator of the first element that is thrown at the first time.
  6. So, If it is not sorted, the result of uniquified vector is not the same as our thought. (you can check the result above)

Solution


{...}

int main(void) {
  vector<int> v = {1, 1, 1, 5, 4, 2, 3, 1, 1};

  sort(v.begin(), v.end());
  v.erase(unique(v.begin(), v.end()), v.end());
  print(v);
  // result: 1 2 3 4 5

  return 0;
}
  1. So first, we sort the vector using C++ STL sort function or stable_sort or ETC...
  2. Then, throw duplicated items to the end of the vector.
  3. Then, get the iterator of the first element that is thrown at the first time.
  4. Then, erase the elements from the returned iterator to the end of the vector.
  5. Finally, print the result.

Other Option

#include <set>
#include <unordered_set>

{...}

int main(void) {
  set<int> si;
  unordered_set<int> usi;
  vector<int> v = {1, 1, 1, 5, 4, 2, 3, 1, 1};

  for (const auto& val : v) {
    si.insert(val);
    usi.insert(val);
  }
  print(si);
  // result: 1 2 3 4 5(result is sorted)
  print(usi);
  // result: 1 5 4 2 3(result is not sorted)

  return 0;
}
  1. Use set or unordered_set to remove duplicated elements.
  2. But, unordered_set could not be used with lower_bound which is used for binary search.
  3. So, If you need to search elements in algorithm problem, you should use set or unique.
